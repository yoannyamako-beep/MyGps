# GPS Guidage Dynamique

## Présentation
Système de guidage GPS capable de calculer et recalculer dynamiquement le meilleur chemin dans un réseau routier en fonction du trafic et des accidents.

## Fonctionnalités
- Choix du départ et de la destination
- Calcul du meilleur itinéraire avec Dijkstra
- Recalcul dynamique en cas d’accident
- Interface graphique simple et interactive

## Technologies utilisées
- Java 17
- Swing pour l’interface graphique
- Programmation orientée objet et événementielle

## Instructions
1. Compiler : `javac src/**/*.java`
2. Lancer : `java src.main`
